# Python コーディング規約

## 文字コードと記号
- 全角括弧（（、））は使用せず、半角括弧（(、)）を使用する
- 全角コロン（：）は使用せず、半角コロン（:）を使用する
- コメントやdocstring内でも同様に半角記号を使用する

## インポート
- すべてのimport文はファイルのトップレベルに配置する
- 関数内や条件分岐内でのimportは避ける

## パス操作
- `os.path.exists()`の代わりに`Path.exists()`を使用する
- `open()`の代わりに`Path.open()`を使用する

## 例外処理
- tryブロック内でreturnする場合は、elseブロックを使用する
- 不要なコメントアウトされたコードは削除する

## コード構造
- ネストしたif文は可能な限り単一のif文に結合する
- 関数は50ステートメント以下、分岐は12以下を目指す

## 型ヒント
- 可能な限り型ヒントを使用する
